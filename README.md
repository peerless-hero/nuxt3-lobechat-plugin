# Nuxt3 LobeChat Plugin

这是一个基于 Nuxt3 的 LobeChat 插件项目。

## 安装

请按照以下步骤安装和配置项目：

1. 克隆仓库：

   ```bash
   git clone https://github.com/yourusername/nuxt3-lobechat-plugin.git
   ```

2. 进入项目目录：

   ```bash
    cd nuxt3-lobechat-plugin
   ```

3. 安装依赖：

   ```bash
   npm install
   ```

## 使用

启动开发服务器：

```bash
npm run dev
```

构建生产版本：

```bash
npm run build
```

启动生产服务器：

```bash
npm run start
```
